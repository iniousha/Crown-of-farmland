this is a text-based strategy game implemented in Java sd part of the "programmieren" course at Karlsruhe Institute of Technology (KIT). 
it features:
interactive command-line interface
7X7 board system with costum rendering 
hidden and revealed units
unit movement and blocking mechanics 
duel and damage calculation system 
unit merging/ compatibility mechanics 
AI-constrolled opponent 
multiple board display modes
deterministic randomness using seeds
example commands:
select D3
move D4
place 2
place 2 3
flip 
block 
board
state 
yield 
quit
this project mainly focused on: 
object orianted systems
how to implement complex game logic 
parsing sructured files
strict checkstyle requirements 
handling interactive command-line input
